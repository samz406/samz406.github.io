еҸ‘иЎЁдәҺ 2025-03-24 еҲҶзұ»дәҺ AI

жЈҖзҙўеўһејәз”ҹжҲҗпјҲRetrieval-Augmented GenerationпјҢз®Җз§°RAGпјүжҠҖжңҜжӯЈйҖҗжёҗжҲҗдёәи§ЈеҶіеӨ§еһӢиҜӯиЁҖжЁЎеһӢпјҲLLMпјүзҹҘиҜҶеұҖйҷҗжҖ§зҡ„йҮҚиҰҒж–№жі•

RAGжҠҖжңҜжҰӮиҝ°

RAGзҡ„е®ҡд№үдёҺж ёеҝғжҖқжғі

RAGжҳҜдёҖз§Қе°ҶжЈҖзҙўзі»з»ҹдёҺз”ҹжҲҗејҸAIжЁЎеһӢзӣёз»“еҗҲзҡ„жҠҖжңҜжЎҶжһ¶пјҢе…¶ж ёеҝғжҖқжғіжҳҜеңЁз”ҹжҲҗе“Қеә”еүҚпјҢе…Ҳд»ҺеӨ–йғЁзҹҘиҜҶеә“дёӯжЈҖзҙўзӣёе…ідҝЎжҒҜпјҢз„¶еҗҺе°ҶиҝҷдәӣдҝЎжҒҜдҪңдёәдёҠдёӢж–Үиҫ“е…ҘеҲ°з”ҹжҲҗжЁЎеһӢдёӯпјҢд»ҺиҖҢеўһејәжЁЎеһӢзҡ„иҫ“еҮәиҙЁйҮҸгҖӮиҝҷз§Қж–№жі•жңүж•Ҳең°и§ЈеҶідәҶLLMйқўдёҙзҡ„еҮ дёӘе…ій”®й—®йўҳпјҡзҹҘиҜҶж—¶ж•ҲжҖ§йҷҗеҲ¶гҖҒдё“дёҡйўҶеҹҹзҹҘиҜҶдёҚи¶ігҖҒд»ҘеҸҠе№»и§үпјҲз”ҹжҲҗдёҚеҮҶзЎ®дҝЎжҒҜпјүзӯүгҖӮ

RAGзҡ„жҠҖжңҜжј”иҝӣ

RAGжҠҖжңҜжңҖж—©з”ұMetaеңЁ2020е№ҙйҖҡиҝҮи®әж–ҮгҖҠRetrieval-Augmented Generation for Knowledge-Intensive NLP TasksгҖӢжӯЈејҸжҸҗеҮәгҖӮйҡҸзқҖеӨ§иҜӯиЁҖжЁЎеһӢзҡ„еҝ«йҖҹеҸ‘еұ•пјҢRAGжҠҖжңҜд№ҹеңЁдёҚж–ӯжј”иҝӣпјҡ

  • дј з»ҹRAGпјҡз®ҖеҚ•ең°е°ҶжЈҖзҙўз»“жһңдҪңдёәpromptзҡ„дёҖйғЁеҲҶиҫ“е…ҘеҲ°LLM
  • й«ҳзә§RAGпјҡеј•е…ҘдәҶеӨҡжӯҘжЈҖзҙўгҖҒйҮҚж–°жҺ’еәҸгҖҒдёҠдёӢж–ҮеҺӢзј©зӯүдјҳеҢ–жҠҖжңҜ
  • иҮӘйҖӮеә”RAGпјҡиғҪеӨҹж №жҚ®й—®йўҳзұ»еһӢеҠЁжҖҒи°ғж•ҙжЈҖзҙўзӯ–з•Ҙе’ҢеҸӮж•°

RAGдёҺзәҜLLMж–№жЎҲзҡ„жҜ”иҫғ

зӣёжҜ”зәҜзІ№дҫқиө–LLMеҸӮж•°дёӯеӯҳеӮЁзҡ„зҹҘиҜҶпјҢRAGе…·жңүеӨҡйЎ№жҳҫи‘—дјҳеҠҝпјҡ

зү№жҖ§ зәҜLLM RAG
зҹҘиҜҶжӣҙж–° йңҖиҰҒйҮҚж–°и®ӯз»ғжҲ–еҫ®и°ғ еҸӘйңҖжӣҙж–°зҹҘиҜҶеә“
дё“дёҡзҹҘиҜҶ жңүйҷҗдё”йҡҫд»Ҙжү©еұ• еҸҜиҪ»жқҫйӣҶжҲҗдё“дёҡйўҶеҹҹж–ҮжЎЈ

иҮӘе®ҡд№үжҖ§ жңүйҷҗ й«ҳеәҰеҸҜе®ҡеҲ¶

RAGзҡ„жҠҖжңҜжһ¶жһ„

RAGзі»з»ҹзҡ„дё»иҰҒз»„д»¶

дёҖдёӘе®Ңж•ҙзҡ„RAGзі»з»ҹйҖҡеёёеҢ…еҗ«д»ҘдёӢж ёеҝғз»„д»¶пјҡ

  1. ж–ҮжЎЈеҠ иҪҪеҷЁпјҡиҙҹиҙЈд»Һеҗ„з§ҚжқҘжәҗиҺ·еҸ–е’Ңйў„еӨ„зҗҶж–ҮжЎЈ
  2. ж–ҮжЎЈеҲҮеҲҶеҷЁпјҡе°Ҷй•ҝж–ҮжЎЈеҲҶеүІжҲҗйҖӮеҗҲжЈҖзҙўзҡ„е°Ҹеқ—
  3. еҗ‘йҮҸеҢ–жЁЎеқ—пјҡе°Ҷж–Үжң¬иҪ¬жҚўдёәиҜӯд№үеҗ‘йҮҸиЎЁзӨә
  4. еҗ‘йҮҸж•°жҚ®еә“пјҡеӯҳеӮЁе’Ңзҙўеј•ж–Үжң¬еҗ‘йҮҸпјҢж”ҜжҢҒзӣёдјјжҖ§жҗңзҙў
  5. жЈҖзҙўеҷЁпјҡж №жҚ®з”ЁжҲ·жҹҘиҜўжүҫеҲ°зӣёе…іж–ҮжЎЈ
  6. з”ҹжҲҗжЁЎеһӢпјҡз»“еҗҲжЈҖзҙўз»“жһңз”ҹжҲҗжңҖз»Ҳеӣһзӯ”
  7. иҜ„дј°дёҺдјҳеҢ–жЁЎеқ—пјҡиҜ„дј°зі»з»ҹжҖ§иғҪе№¶дёҚж–ӯдјҳеҢ–

жЈҖзҙўжңәеҲ¶иҜҰи§Ј

жЈҖзҙўжҳҜRAGзі»з»ҹзҡ„е…ій”®зҺҜиҠӮпјҢдё»иҰҒжңүд»ҘдёӢеҮ з§Қе®һзҺ°ж–№ејҸпјҡ

  • зЁҖз–ҸжЈҖзҙўпјҡеҹәдәҺе…ій”®иҜҚеҢ№й…Қзҡ„дј з»ҹжЈҖзҙўж–№жі•пјҢеҰӮBM25з®—жі•
  • еҜҶйӣҶжЈҖзҙўпјҡеҹәдәҺиҜӯд№үеҗ‘йҮҸзҡ„зӣёдјјжҖ§жҗңзҙў
  • ж··еҗҲжЈҖзҙўпјҡз»“еҗҲзЁҖз–ҸжЈҖзҙўе’ҢеҜҶйӣҶжЈҖзҙўзҡ„дјҳеҠҝ
  • еӨҡжҹҘиҜўжЈҖзҙўпјҡз”ҹжҲҗеӨҡдёӘзӣёе…іжҹҘиҜўд»ҘиҺ·еҸ–жӣҙе№ҝжіӣзҡ„зӣёе…іж–ҮжЎЈ
  • йҖ’еҪ’жЈҖзҙўпјҡйҖҡиҝҮеӨҡиҪ®жЈҖзҙўйҖҗжӯҘз»ҶеҢ–з»“жһң

еҗ‘йҮҸж•°жҚ®еә“дёҺеҗ‘йҮҸжҗңзҙў

еҗ‘йҮҸж•°жҚ®еә“жҳҜRAGзі»з»ҹзҡ„ж ёеҝғеӯҳеӮЁз»„д»¶пјҢеёёи§Ғзҡ„ејҖжәҗе’Ңе•ҶдёҡйҖүжӢ©еҢ…жӢ¬пјҡ

  • ејҖжәҗйҖүйЎ№пјҡChroma, FAISS, Milvus, Weaviate
  • е•ҶдёҡйҖүйЎ№пјҡPinecone, Qdrant, Vespa

иҝҷдәӣеҗ‘йҮҸж•°жҚ®еә“йҖҡеёёж”ҜжҢҒеӨҡз§Қзҙўеј•зұ»еһӢе’Ңиҝ‘дјјжңҖиҝ‘йӮ»пјҲANNпјүз®—жі•пјҢеҰӮHNSWпјҲHierarchical Navigable Small WorldпјүгҖҒIVFпјҲInverted File IndexпјүзӯүпјҢд»Ҙе®һзҺ°й«ҳж•Ҳзҡ„зӣёдјјжҖ§жҗңзҙўгҖӮ

RAGзі»з»ҹзҡ„жһ„е»әжөҒзЁӢ

ж•°жҚ®еҮҶеӨҮдёҺйў„еӨ„зҗҶ

  1. ж•°жҚ®ж”¶йӣҶпјҡд»Һеҗ„з§ҚжқҘжәҗпјҲеҰӮPDFж–ҮжЎЈгҖҒзҪ‘йЎөгҖҒж•°жҚ®еә“зӯүпјүж”¶йӣҶзӣёе…іж•°жҚ®
  2. ж•°жҚ®жё…жҙ—пјҡз§»йҷӨдёҚеҝ…иҰҒзҡ„ж јејҸгҖҒж Үи®°е’ҢеҷӘеЈ°
  3. ж–Үжң¬жҸҗеҸ–пјҡд»Һйқһж–Үжң¬ж јејҸпјҲеҰӮPDFгҖҒеӣҫзүҮпјүдёӯжҸҗеҸ–ж–Үжң¬еҶ…е®№
  4. иҜӯиЁҖжЈҖжөӢдёҺиҝҮж»ӨпјҡиҜҶеҲ«ж–ҮжЎЈиҜӯиЁҖе№¶иҝҮж»ӨдёҚзӣёе…іеҶ…е®№

ж–ҮжЎЈеҲҶеқ—зӯ–з•Ҙ

еҲҶеқ—зӯ–з•Ҙдё»иҰҒжңүд»ҘдёӢдёүдёӘе…ій”®з»„жҲҗйғЁеҲҶжһ„жҲҗпјҡ

еӨ§е°ҸпјҡжҜҸдёӘж–ҮжЎЈеқ—жүҖе…Ғи®ёзҡ„жңҖеӨ§еӯ—з¬Ұж•°гҖӮ

йҮҚеҸ пјҡеңЁзӣёйӮ»ж•°жҚ®еқ—д№Ӣй—ҙпјҢйҮҚеҸ еӯ—з¬Ұзҡ„ж•°йҮҸгҖӮ

жӢҶеҲҶпјҡйҖҡиҝҮж®өиҗҪиҫ№з•ҢгҖҒеҲҶйҡ”з¬ҰгҖҒж Үи®°пјҢжҲ–иҜӯд№үиҫ№з•ҢжқҘзЎ®е®ҡеқ—иҫ№з•Ңзҡ„дҪҚзҪ®гҖӮ

дёҠиҝ°дёүдёӘз»„жҲҗйғЁеҲҶе…ұеҗҢеҶіе®ҡдәҶеҲҶеқ—зӯ–з•Ҙзҡ„зү№жҖ§еҸҠе…¶йҖӮз”ЁеңәжҷҜгҖӮеҹәдәҺиҝҷдәӣз»„жҲҗйғЁеҲҶпјҢеёёи§Ғзҡ„еҲҶеқ—зӯ–з•ҘеҢ…жӢ¬пјҡеӣәе®ҡеӨ§е°ҸеҲҶеқ—пјҲFixed Size ChunkingпјүгҖҒйҮҚеҸ еҲҶеқ—пјҲOverlap ChunkingпјүгҖҒйҖ’еҪ’еҲҶеқ—пјҲRecursive ChunkingпјүгҖҒж–ҮжЎЈзү№е®ҡеҲҶеқ—пјҲDocument Specific ChunkingпјүгҖҒиҜӯд№үеҲҶеқ—пјҲSemantic ChunkingпјүгҖҒж··еҗҲеҲҶеқ—пјҲMix ChunkingпјүгҖӮ

жЈҖзҙўзӯ–з•Ҙи®ҫи®Ў

жңүж•Ҳзҡ„жЈҖзҙўзӯ–з•ҘйҖҡеёёеҢ…еҗ«д»ҘдёӢеҮ дёӘзҺҜиҠӮпјҡ

  1. жҹҘиҜўеӨ„зҗҶпјҡеҜ№з”ЁжҲ·жҹҘиҜўиҝӣиЎҢеҲҶжһҗе’ҢеӨ„зҗҶ
  2. жҹҘиҜўжү©еұ•пјҡжү©е……еҺҹе§ӢжҹҘиҜўд»ҘиҰҶзӣ–жӣҙеӨҡзӣёе…ідҝЎжҒҜ
  3. еӨҡи·ҜжЈҖзҙўпјҡеҗҢж—¶дҪҝз”ЁеӨҡз§ҚжЈҖзҙўж–№жі•е№¶еҗҲе№¶з»“жһң
  4. з»“жһңйҮҚжҺ’еәҸпјҡж №жҚ®зӣёе…іжҖ§гҖҒж–°йў–жҖ§зӯүеӣ зҙ еҜ№жЈҖзҙўз»“жһңиҝӣиЎҢйҮҚж–°жҺ’еәҸ
  5. дёҠдёӢж–Үз»„иЈ…пјҡе°ҶжЈҖзҙўеҲ°зҡ„ж–ҮжЎЈзүҮж®өз»„иЈ…жҲҗиҝһиҙҜзҡ„дёҠдёӢж–Ү

з”ҹжҲҗжЁЎеһӢдёҺжҸҗзӨәе·ҘзЁӢ

RAGзі»з»ҹдёӯзҡ„з”ҹжҲҗзҺҜиҠӮдё»иҰҒж¶үеҸҠдёӨдёӘе…ій”®еӣ зҙ пјҡ

  1. LLMйҖүжӢ©пјҡж №жҚ®йңҖжұӮйҖүжӢ©еҗҲйҖӮзҡ„иҜӯиЁҖжЁЎеһӢпјҢеҰӮGPTгҖҒClaudeгҖҒLlamaгҖҒDeepSeak,Qwenзӯү
  2. жҸҗзӨәе·ҘзЁӢпјҡи®ҫи®Ўй«ҳж•Ҳзҡ„жҸҗзӨәжЁЎжқҝпјҢжҢҮеҜјжЁЎеһӢеҰӮдҪ•дҪҝз”ЁжЈҖзҙўеҲ°зҡ„дҝЎжҒҜ

дёҖдёӘиүҜеҘҪзҡ„RAGжҸҗзӨәжЁЎжқҝйҖҡеёёеҢ…еҗ«д»ҘдёӢз»„д»¶пјҡ

  • зі»з»ҹжҢҮд»Өпјҡе®ҡд№үжЁЎеһӢзҡ„иЎҢдёәе’Ңеӣһзӯ”ж–№ејҸ
  • жЈҖзҙўз»“жһңпјҡжҸҗдҫӣзӣёе…ізҡ„еӨ–йғЁзҹҘиҜҶ
  • з”ЁжҲ·й—®йўҳпјҡжҳҺзЎ®з”ЁжҲ·зҡ„е…·дҪ“йңҖжұӮ
  • ж јејҸжҢҮзӨәпјҡи§„е®ҡеӣһзӯ”зҡ„ж јејҸе’Ңз»“жһ„

RAGзҡ„еә”з”ЁеңәжҷҜ

дјҒдёҡзҹҘиҜҶеә“

  • еҝ«йҖҹеӣһзӯ”е‘ҳе·Ҙе…ідәҺе…¬еҸёж”ҝзӯ–гҖҒжөҒзЁӢзҡ„й—®йўҳ
  • жҠҖжңҜж”ҜжҢҒе’Ңй—®йўҳжҺ’жҹҘзҡ„жҷәиғҪиҫ…еҠ©
  • еӨ§йҮҸеҶ…йғЁж–ҮжЎЈзҡ„иҜӯд№үжҗңзҙўе’ҢдҝЎжҒҜжҸҗеҸ–
  • ж–°е‘ҳе·Ҙеҹ№и®ӯе’ҢзҹҘиҜҶдј жүҝ

еһӮзӣҙйўҶеҹҹдё“е®¶зі»з»ҹ

еңЁеҢ»з–—гҖҒжі•еҫӢгҖҒйҮ‘иһҚзӯүдё“дёҡйўҶеҹҹпјҡ

  • еҢ»з–—иҜҠж–ӯиҫ…еҠ©е’Ңз—…дҫӢеҲҶжһҗ
  • жі•еҫӢж–Үд»¶жЈҖзҙўе’ҢеҗҲи§„е®ЎжҹҘ
  • йҮ‘иһҚеёӮеңәж•°жҚ®еҲҶжһҗе’ҢжҠ•иө„е»әи®®
  • з§‘з ”ж–ҮзҢ®з»јиҝ°е’Ңе…іиҒ”еҲҶжһҗ

е®ўжҲ·жңҚеҠЎдёҺж”ҜжҢҒ

  • дә§е“ҒFAQжҷәиғҪй—®зӯ”
  • жҠҖжңҜж”ҜжҢҒз–‘йҡҫй—®йўҳи§Јзӯ”
  • е®ўжҲ·еҸҚйҰҲеҲҶжһҗе’ҢиҮӘеҠЁеӣһеӨҚ
  • еӨҡиҜӯиЁҖе®ўжҲ·ж”ҜжҢҒ

дёӘдәәзҹҘиҜҶз®ЎзҗҶ

  • жһ„е»әдёӘдәәеӯҰд№ з¬”и®°еә“
  • з®ЎзҗҶе’ҢжЈҖзҙўдёӘдәәж–ҮжЎЈ
  • еҲӣдҪңиҫ…еҠ©е’Ңз ”з©¶ж”ҜжҢҒ
  • дёӘжҖ§еҢ–дҝЎжҒҜеҠ©жүӢ
