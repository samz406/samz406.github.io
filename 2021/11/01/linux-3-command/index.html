Linux 三剑客简介

发表于 2021-11-01 分类于 技术

Linux 三剑客是指在 Linux 环境中非常有用且常用的三个命令行工具— grep，awk 和 sed。这三个工具能力强大，是复杂文件处理和数据分析不可或缺的工具。下面我们来详细了解这三个工具和它们的使用场景。

  • grep：专注于搜索和匹配。
  • awk：擅长分析和处理结构化数据。
  • sed：强大的流式文本编辑器，适合内容修改。

grep

grep是Global Regular Expression Print的缩写，用于根据正则表达式在文件中搜索相关内容并打印匹配的行。这个工具应用场景非常广泛，特别适合选择和过滤大量文件内容。

基本用法

以下是一些常见示例：

  1. 在文件中搜索关键词：

    在指定文件中查找包含“keyword”的行。

  2. 递归搜索目录中的关键词：

    在指定目录及其子目录下递归查找包含“keyword”的文件。

  3. 显示仅匹配部分：

    打印文件中所有长度大于等于 3 的单词。

  4. 忽略大小写搜索：

    不区分大小写地匹配关键词。

  5. 结合管道使用：

    将其他命令的输出作为输入，通过 grep 搜索关键词。

awk

awk是一种功能强大的文本处理工具，其名称来源于创始人 Alfred Aho，Peter Weinberger 和 Brian Kernighan 的姓氏首字母。它可以按照模式匹配处理文本并执行各种操作，特别擅长处理结构化数据。

基本用法

  1. 打印指定列：

    打印文件中的第一列和第三列。

  2. 根据模式匹配过滤并打印：

    匹配包含“pattern”的行并打印整行。

  3. 计算总和：

    累加文件第二列的值，并在文件末尾输出总和。

  4. 指定字段分隔符：

    使用冒号作为分隔符来解析文件，并打印第一列和第二列。

  5. 多条件处理：

    处理满足多个条件的行，例如第二列大于 50 且第三列等于 “PASS” 的行。

  6. 动态字段处理：

    遍历每一行的所有字段并逐一打印。

sed

sed是“stream editor”（流式编辑器）的缩写。它擅长处理文件流的内容修改，例如文本替换、删除、插入等操作。它是 Linux 中不可或缺的文本编辑工具。

基本用法

  1. 字符串替换：

    将文件中所有“old”替换为“new”。

  2. 删除匹配的行：

    删除文件中匹配“pattern”的行。

  3. 在匹配行前后插入内容：

    在匹配行之前插入“New content”。

    在匹配行之后插入“New content”。

  4. 打印指定范围的行：

    打印文件的第 10 到第 20 行。

  5. 结合管道处理数据流：

    将其他命令的输出传递给 sed 进行处理。

  6. 只打印匹配的行：

    只输出匹配“pattern”的行。
