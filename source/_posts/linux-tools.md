 linux 常用命令介绍

#### top

| 参数 | 说明                                                         |
| ---- | ------------------------------------------------------------ |
| PR   |                                                              |
| NI   |                                                              |
| VIRT | 进程虚拟内存，只要是进程申请过的内存，即便还没有真正分配物理内存，也会计算在内 |
| RES  | 进程实际使用物理内存                                         |
| SHR  | 共享内存大小，他进程共同使用的共享内存、加载的动态链接库以及程序的代码段等 |
| %MEM | 进程使用物理内存占系统总内存的百分比                         |



#### pidstat



| 参数      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| kB_rd/s   | 每秒读取的数据大小                                           |
| kB_wr/s   | 每秒发出的写请求数据大小                                     |
| kB_ccwr/s | 每秒取消的写请求数据大小                                     |
| iodelay   | 块 I/O 延迟（iodelay），包括等待同步块 I/O 和换入块 I/O 结束的时间，单位是时钟周期。 |







#### iostat

| 参数     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| r/s      | 每秒发送给磁盘读请求数                                       |
| w/s      | 每秒发送给磁盘写请求数                                       |
| rkB/s    | 每秒从磁盘读取的数据量                                       |
| wkB/s    | 每秒向磁盘写入的数据量                                       |
| rrqm/s   | 每秒合并的读请求                                             |
| wrqm/s   | 每秒合并的写请求                                             |
| r_await  | 读请求处理完成等待时间（队列中等待时间和设备实际处理时间（毫秒） |
| w_await  | 写请求处理完成等待时间，（队列等待时间和设备实际处理时间（毫秒）） |
| aqu-sz   | 平均排队大小(kB)                                             |
| rareq-sz | 平均读请求大小（kB）                                         |
| wareq-sz | 平均写请求大小（kB）                                         |
| svctm    | 处理I/O请求所需要的平均时间，单位毫秒。                      |
| %util    | 磁盘使用率，磁盘处理I/O时间百分比，由于存在并行I/O，100%也不定是磁盘饱和 |





#### vmstat

进程、内存、分页、块IO、陷阱、磁盘和cpu活动的信息  

-d 查看磁盘统计

-m slab信息

-w  输出更多



| 分类   | 参数   | 说明                         |
| ------ | ------ | ---------------------------- |
| Procs  | r      | 运行中的进程或等待运行的进程 |
| Procs  | b      | 阻塞的进程                   |
| Memory | swpd   | 虚拟内存                     |
| Memory | free   | 空闲内存                     |
| Memory | buff   | 内存作为buff                 |
| Memory | cache  | 内存作为cache                |
| Memory | inact  | 没有活动内存的数量           |
| Memory | active | 活动内存的数量               |
| swap   | si     | 从磁盘交换入的内存量（/s）   |
| swap   | so     | 交换到磁盘的内存数量(/s)     |
| IO     | bi     | 从设备上接收数据             |
| IO     | bo     | 发送给块设备                 |
| System | in     | 每秒中断次数                 |
| System | cs     | 没秒上下文切换次数           |
| CPU    | us     | 用户态使用时间               |
| CPU    | sy     | 内核态时间                   |
| CPU    | id     | 空闲时间                     |
| CPU    | wa     | 等待IO花费的时间             |
| CPU    | st     | 虚拟机使用时间               |

#### free

| 参数       | 说明                         |
| ---------- | ---------------------------- |
| total      | 总内存大小                   |
| used       | 已使用内的大小，包括共享内存 |
| free       | 未使用内存大小               |
| shared     | 共享内存大小                 |
| buff/cache | 缓存和缓冲区                 |
| available  | 新进程可用内存               |

> available 包含未使用内存和可回收的缓存，一般会比未使用内存更大
