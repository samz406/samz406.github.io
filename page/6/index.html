创新扩散理论通俗研究报告

发表于 2025-04-10 分类于 研究报告

​ 在人类社会中，新技术、新理念的传播往往遵循着某种规律：有的人率先尝试新品，有的人则观望许久才跟随，还有一部分人甚至直到身边的人几乎都使用了才慢慢接受。为什么会出现这种现象？为什么有些创新会像野火般迅速蔓延，而另一些创新却推广困难、进展缓慢？创新扩散理论（Diffusion of Innovations Theory）正是用来解释这些问题的一个经典理论框架。这一理论由美国社会学家兼传播学者埃弗雷特·M·罗杰斯（Everett M. Rogers）于1962年提出，自问世以来在农业、教育、医疗、商业管理、科技推广等诸多领域得到广泛应用。通俗地说，创新扩散理论关注的就是“新事物是如何被人们逐步接受并推广开来的”。本报告将全面介绍创新扩散理论的起源和发展、核心概念、创新采纳的阶段过程、不同类型采纳者的特征、影响扩散的关键因素，以及理论在现实中的应用案例。同时，我们也将讨论对该理论的批评和后续发展。希望通过通俗易懂的阐述，帮助读者了解这一影响深远的理论。

Transformer浅析

发表于 2025-04-08 分类于 AI

$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$

这个是 Transformer 中最核心的 Attention 公式，其中：

  • ( Q )：Query 查询向量
  • ( K )：Key 键向量
  • ( V )：Value 值向量

Q 是由用户输入的 token embedding 经过线性变换得到的，也就是说：

Q 不是“原始的 token”，而是经过 Embedding 和线性投影后，专门用于“查询”的向量。

假设用户输入了这句话：

User: “fluffy creature”

经过：

  1. Tokenization → 变成 token（比如两个词）
  2. Embedding → 变成高维向量（比如每个是 768 维）

然后这些向量会分别乘以一个专门的矩阵，得到：

其中 ( W_Q, W_K, W_V ) 是不同的可学习权重矩阵，用来生成不同的向量空间表示。

向量 作用 来源
Q（Query） 表示“我想关注什么” 每个输入 token × ( W_Q )
K（Key） 表示“我能提供什么信息” 每个输入 token × ( W_K )
V（Value） 表示“我的内容值是多少” 每个输入 token × ( W_V )

而 Attention 的核心就在于：

用 Query 去“打分”所有的 Key，得到权重分布（softmax），再加权平均对应的 Value。

你可以把这整个过程类比成：

用户问一个问题（Q），然后去“数据库中的每条记录（K）”打分，找出相关记录，再取出这些记录的内容（V）做加权平均。
