Transformer浅析

发表于 2025-04-08 分类于 AI

$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$

这个是 Transformer 中最核心的 Attention 公式，其中：

  • ( Q )：Query 查询向量
  • ( K )：Key 键向量
  • ( V )：Value 值向量

Q 是由用户输入的 token embedding 经过线性变换得到的，也就是说：

Q 不是“原始的 token”，而是经过 Embedding 和线性投影后，专门用于“查询”的向量。

假设用户输入了这句话：

User: “fluffy creature”

经过：

  1. Tokenization → 变成 token（比如两个词）
  2. Embedding → 变成高维向量（比如每个是 768 维）

然后这些向量会分别乘以一个专门的矩阵，得到：

其中 ( W_Q, W_K, W_V ) 是不同的可学习权重矩阵，用来生成不同的向量空间表示。

向量 作用 来源
Q（Query） 表示“我想关注什么” 每个输入 token × ( W_Q )
K（Key） 表示“我能提供什么信息” 每个输入 token × ( W_K )
V（Value） 表示“我的内容值是多少” 每个输入 token × ( W_V )

而 Attention 的核心就在于：

用 Query 去“打分”所有的 Key，得到权重分布（softmax），再加权平均对应的 Value。

你可以把这整个过程类比成：

用户问一个问题（Q），然后去“数据库中的每条记录（K）”打分，找出相关记录，再取出这些记录的内容（V）做加权平均。

人类意识探索调研报告

发表于 2025-03-25 分类于 研究报告

意识是指个体对自身内部状态和外部环境的觉知或察觉，是人类心灵最基本却又最神秘的现象之一。数千年来，哲学家、科学家和神学家对意识的本质提出了各种分析和解释，但至今仍未有统一定论 。简单来说，当我们处于有意识的心理状态时，我们能够从第一人称视角感受到“某种体验的样子”——例如闻到玫瑰香味或看到绚丽色彩时，那种主观的感受。这种主观体验之谜使意识问题成为当代哲学和科学领域最具挑战性的议题之一。

意识研究具有显著的跨学科特性。哲学上，意识涉及心灵与物质的关系等基本问题；心理学上，意识关联注意、感知、认知等心理功能；神经科学上，意识被视为大脑活动的结果 ；人工智能领域则探讨机器是否能拥有类人的主观体验；宗教和精神传统中，意识常被赋予灵性意义。不同领域对意识的定义和理解各不相同，使其概念既丰富又复杂。尽管近年来科学技术的发展使我们对大脑和行为的认识不断加深，人类对于“神经活动如何产生主观体验”这一问题的理解仍然十分有限 。
